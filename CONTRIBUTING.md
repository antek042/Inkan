# Contributing Guidelines

Thank you for your interest in contributing to Inkan!


## How can I contribute?

- Report bugs via [Issues](../../issues).
- Suggest new features.
- Submit pull requests with fixes or improvements.

## Code style

- Use [Black](https://github.com/psf/black) for code formatting:
  ```bash
  black .
  ```
- All tests must pass before submitting a PR.

## Pull request process

1. Fork the repository.
2. Create a new branch:
   ```bash
   git checkout -b feature/your-feature-name
   ```
3. Make your changes and ensure code passes formatting/tests.
4. Commit with a clear and descriptive message.
5. Push your branch and open a Pull Request.
